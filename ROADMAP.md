# Roadmap

## Near term

- Richer matchers: regex, exact normalized stdout, and file assertions.
- Safer command presets for common package managers.
- Better generated contracts from Markdown docs and examples folders.
- JUnit output for CI systems.

## Later

- Optional container runner interface.
- Contract update mode for approved golden output changes.
- VS Code snippets and schema hints.

## Non-goals

- Hosted dashboard.
- Hidden telemetry.
- Secret collection.
